# 守界者贡献指南

欢迎你，新的守界者！感谢你愿意为加固这道“宇宙最后的防线”贡献一份力量。请遵循以下指南，让我们的协作更加高效。

## 🌟 如何贡献

你可以通过以下方式成为守界者：

1.  **修正与改进**：修复错别字、改善措辞、优化文档结构。
2.  **理论深化**：撰写分析文章、解决理论中的矛盾、推导新的结论。
3.  **内容创作**：创作新的文明、角色、故事、艺术设计等。
4.  **工具开发**：制作可视化图表、计算工具、网站等。
5.  **社区建设**：帮助回答问题、整理讨论、组织活动。

## 🛠️ 贡献流程

1.  **Fork 本仓库**
2.  **创建你的分支** (`git checkout -b feature/你的功能名`)
3.  **提交你的更改** (`git commit -m '添加了某个精彩的功能'`)
4.  **推送到分支** (`git push origin feature/你的功能名`)
5.  **开启一个 Pull Request**

## 📝 创作规范

- **尊重核心设定**：请确保你的创作不与 [核心理论体系](/核心理论体系/) 中的元设定（如存在张力、虚无本源、文明熵增悖论）相冲突。
- **保持风格统一**：使用简洁、清晰的中文，与原文的哲学思辨风格相协调。
- **注明来源**：如果你的创作引用了官方或其他社区成员的内容，请务必注明。
- **文件命名**：使用清晰的英文或拼音文件名，例如 `wen-ming-jue-ce-liu-tiao-lu.md`。

## ❓ 需要灵感？

- 去看看 [`🌐 社区与协作/`](/社区与协作/) 下的议题和项目。
- 在 [GitHub Discussions](链接到Discussions) 中提出你的想法，和大家一起讨论。
